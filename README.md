# Quik Wallet Homework

## Prerequisites 

- docker-compose >=1.28.5
- Docker Engine - Community >= 20.10.3
- go1.16

## Configuration

Edit `.env` file to configure the app when needed.

## Build 

```shell
docker-compose up --build
```

## Test

```shell
docker-compose up --build quikwallet_test
```
